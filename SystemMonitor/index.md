# VRC System Monitor
## English
System Monitor that can be added into a VRChat Avatar.
Grab left hand fist to activate overlay and open hand to disable overlay.
## Korean
VRChat 아바타에 추가할 수 있는 시스템 모니터 입니다.
왼손 주먹으로 오버레이를 활성화하고 손을 펴서 오버레이를 끌 수 있습니다.
